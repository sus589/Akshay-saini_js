(ye topic interview ka favourite hai 💯)

🔹 this kya hota hai?

👉 this ek keyword hai jo current execution context ko refer karta hai

👉 Simple line:

"this ka matlab depend karta hai ki function kaise call hua hai"

🔥 Golden Rule

👉 this = jis object ne function call kiya

🔹 1. Global Scope me this
console.log(this);

👉 Browser me:

window object
🔹 2. Normal Function me this
function test() {
  console.log(this);
}

test();

👉 Output:

Non-strict mode → window

Strict mode → undefined

🔹 3. Object ke andar this (Most Important)
const user = {
  name: "Susmita",
  greet: function() {
    console.log(this.name);
  }
};

user.greet();

👉 Output:

Susmita

👉 yaha this = user

🔹 4. Arrow Function me this 🔥

👉 Arrow function apna this nahi banata
👉 outer scope ka this use karta hai

const user = {
  name: "Susmita",
  greet: () => {
    console.log(this.name);
  }
};

user.greet();

👉 Output:

undefined

👉 kyunki this global ho gaya

🔹 5. Method ke andar nested function
const user = {
  name: "Susmita",
  greet: function() {
    function inner() {
      console.log(this.name);
    }
    inner();
  }
};

user.greet();

👉 Output:

undefined
✅ Fix (Arrow use karo)
const user = {
  name: "Susmita",
  greet: function() {
    const inner = () => {
      console.log(this.name);
    };
    inner();
  }
};

user.greet();

👉 Output:

Susmita
🔹 6. Constructor Function me this
function Person(name) {
  this.name = name;
}

const p1 = new Person("Susmita");
console.log(p1.name);

👉 Output:

Susmita

👉 this = new object

🔹 7. call(), apply(), bind() 🔥

👉 manually this control kar sakte ho

const user1 = { name: "A" };
const user2 = { name: "B" };

function greet() {
  console.log(this.name);
}

greet.call(user1); // A
greet.call(user2); // B
🔹 Summary Table 🔥
Case	this value
Global	window
Normal function	window / undefined
Object method	object
Arrow function	outer scope
Constructor	new object
🔹 Use kyu hota hai?
✅ 1. Object properties access
this.name
✅ 2. Dynamic context

👉 different objects ke saath same function use

✅ 3. Reusability

👉 call/apply/bind ke through

🔹 Kab use hota hai?

👉 Real projects me:

Angular components (this.data)

Event handlers

OOP (constructor)

API data binding

🔥 Interview Coding Questions
Q1️⃣ Output kya hoga?
const obj = {
  name: "Susmita",
  greet: function() {
    console.log(this.name);
  }
};

obj.greet();

👉 Answer:

Susmita
Q2️⃣ Output kya hoga?
const obj = {
  name: "Susmita",
  greet: () => {
    console.log(this.name);
  }
};

obj.greet();

👉 Answer:

undefined
Q3️⃣ Tricky 🔥
const obj = {
  name: "Susmita",
  greet: function() {
    setTimeout(function() {
      console.log(this.name);
    }, 1000);
  }
};

obj.greet();

👉 Answer:

undefined
✅ Fix
setTimeout(() => {
  console.log(this.name);
}, 1000);
Q4️⃣ call() use
function test() {
  console.log(this.x);
}

const obj = { x: 10 };

test.call(obj);

👉 Answer:

10
Q5️⃣ Constructor
function Person(name) {
  this.name = name;
}

const p = new Person("A");
console.log(p.name);

👉 Answer:

A
🔹 Interview One-Line Answer

👉 Agar interviewer puche:

"What is this keyword?"

👉 bolo:

"this refers to the object that is executing the current function, and its value depends on how the function is called."
