Chalo ab in Promise ke important methods ko ekdum clear Hinglish me real examples ke saath samajhte hain 👇🔥

🔥 1. Promise.all() → sab success chahiye

👉 Rule:

Sab promises resolve hone chahiye

Agar ek bhi fail hua → direct error

✅ Example:
const p1 = Promise.resolve("A");
const p2 = Promise.resolve("B");
const p3 = Promise.resolve("C");

Promise.all([p1, p2, p3])
  .then(res => console.log(res))
  .catch(err => console.log(err));

👉 Output:

["A", "B", "C"]
❌ Agar ek fail ho gaya:
const p1 = Promise.resolve("A");
const p2 = Promise.reject("Error aya");
const p3 = Promise.resolve("C");

Promise.all([p1, p2, p3])
  .then(res => console.log(res))
  .catch(err => console.log(err));

👉 Output:

Error aya

👉 Use case:

Jab sab APIs ka data ek saath chahiye ho (dashboard)

⚡ 2. Promise.race() → jo pehle complete

👉 Rule:

Jo promise sabse pehle complete (resolve/reject) karega → wahi result

✅ Example:
const p1 = new Promise(resolve => setTimeout(() => resolve("Fast"), 1000));
const p2 = new Promise(resolve => setTimeout(() => resolve("Slow"), 3000));

Promise.race([p1, p2])
  .then(res => console.log(res))
  .catch(err => console.log(err));

👉 Output:

Fast

👉 Agar fast wala reject ho gaya:

const p1 = new Promise((_, reject) => setTimeout(() => reject("Error"), 1000));
const p2 = new Promise(resolve => setTimeout(() => resolve("Success"), 3000));

Promise.race([p1, p2])
  .then(res => console.log(res))
  .catch(err => console.log(err));

👉 Output:

Error

👉 Use case:

Timeout handling (jo fast aaye usko lo)

🧾 3. Promise.allSettled() → sabka result chahiye (fail + success)

👉 Rule:

Sab promises complete hone ke baad result milega

Chahe success ho ya fail

✅ Example:
const p1 = Promise.resolve("A");
const p2 = Promise.reject("Error");
const p3 = Promise.resolve("C");

Promise.allSettled([p1, p2, p3])
  .then(res => console.log(res));

👉 Output:

[
  { status: "fulfilled", value: "A" },
  { status: "rejected", reason: "Error" },
  { status: "fulfilled", value: "C" }
]

👉 Use case:

Jab sab APIs ka result dekhna ho (fail bhi important hai)

🎯 4. Promise.any() → first success chahiye

👉 Rule:

Jo pehla resolve karega → wahi result

Sab fail ho gaye → error

✅ Example:
const p1 = Promise.reject("Fail 1");
const p2 = new Promise(resolve => setTimeout(() => resolve("Success"), 2000));
const p3 = Promise.reject("Fail 2");

Promise.any([p1, p2, p3])
  .then(res => console.log(res))
  .catch(err => console.log(err));

👉 Output:

Success
❌ Agar sab fail ho gaye:
const p1 = Promise.reject("Fail 1");
const p2 = Promise.reject("Fail 2");

Promise.any([p1, p2])
  .then(res => console.log(res))
  .catch(err => console.log(err));

👉 Output:

AggregateError

👉 Use case:

Multiple servers se data lena → jo pehle success ho usko use karo

🧠 Easy Comparison (Interview me bolna 🔥)
Method	Kya karta hai
Promise.all()	Sab success chahiye
Promise.race()	Jo pehle complete
Promise.allSettled()	Sabka result (fail + success)
Promise.any()	First success

  .then(res => console.log(res))
  .catch(err => console.log(err));

👉 Output:

Fast Success
