🔹 Higher-Order Function kya hota hai?

👉 Aisa function jo:

Function ko argument me le ✔
ya

Function return kare ✔

👉 use bolte hain Higher-Order Function

🔹 Simple Definition (Interview Line)

"A higher-order function is a function that takes another function as an argument or returns a function."

🔹 Example 1 (Function as Argument)
function greet(name) {
  return "Hello " + name;
}

function processUser(callback) {
  let result = callback("Susmita");
  console.log(result);
}

processUser(greet);

👉 Output:

Hello Susmita

👉 processUser = Higher-Order Function
👉 greet = callback

🔹 Example 2 (Function Return karna)
function multiplier(x) {
  return function(y) {
    return x * y;
  };
}

const double = multiplier(2);

console.log(double(5));

👉 Output:

10

👉 yaha outer function ek aur function return kar raha hai

🔹 Real Life Example 🔥 (Most Important)
Array Methods (sab HOF hai)
1. map()
let arr = [1, 2, 3];

let result = arr.map(function(num) {
  return num * 2;
});

console.log(result);

👉 Output:

[2, 4, 6]
2. filter()
let arr = [1, 2, 3, 4];

let even = arr.filter(function(num) {
  return num % 2 === 0;
});

console.log(even);

👉 Output:

[2, 4]
3. forEach()
let arr = [1, 2, 3];

arr.forEach(function(num) {
  console.log(num);
});

👉 Output:

1
2
3
🔹 Higher-Order Function ka use kyu hota hai?
✅ 1. Code Reusability

👉 ek hi function me different logic pass kar sakte ho

✅ 2. Clean Code

👉 code readable aur short hota hai

✅ 3. Functional Programming

👉 modern JS ka important concept

🔹 Kab use hota hai?

👉 Real projects me:

Array operations (map, filter, reduce)

Event handling

Callbacks

API handling

Angular/React me data transformation

🔹 Simple Real Example
function calculate(a, b, operation) {
  return operation(a, b);
}

const sum = calculate(5, 3, (x, y) => x + y);
const multiply = calculate(5, 3, (x, y) => x * y);

console.log(sum);      // 8
console.log(multiply); // 15
🔹 Interview Coding Questions 🔥
Q1️⃣ Output kya hoga?
function test(fn) {
  return fn(5);
}

console.log(test(function(x) {
  return x * 2;
}));

👉 Answer:

10
Q2️⃣ Output kya hoga?
let arr = [1, 2, 3];

let res = arr.map(num => num + 1);

console.log(res);

👉 Answer:

[2, 3, 4]
Q3️⃣ Filter Question
let arr = [10, 15, 20, 25];

let res = arr.filter(num => num > 15);

console.log(res);

👉 Answer:

[20, 25]
Q4️⃣ Custom Higher-Order Function banao

👉 Question:
Ek HOF banao jo callback use kare aur number cube kare

👉 Solution:

function cube(num, callback) {
  return callback(num);
}

const result = cube(3, function(x) {
  return x * x * x;
});

console.log(result);

👉 Output:

27
Q5️⃣ Tricky 🔥
function outer() {
  return function inner() {
    return "Hello";
  };
}

console.log(outer()());

👉 Answer:

Hello
🔹 Higher-Order vs Callback (Important Difference)
Feature	Higher-Order Function	Callback
Kya hai?	Function	Function
Role	Function ko accept/return karta hai	Pass kiya gaya function
Example	map(), filter()	function inside HOF
