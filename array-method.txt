Chalo map() , filter() , reduce() ko ekdum simple Hinglish + interview level me samajhte hain 👇
(ye 3 sabse important HOF hain 🔥)

🔹 1. map() kya hota hai?

👉 Har element pe operation karke NEW array return karta hai

Example:
let arr = [1, 2, 3];

let result = arr.map(function(num) {
  return num * 2;
});

console.log(result);

👉 Output:

[2, 4, 6]

👉 original array change nahi hota ❌

🔹 Use kab karte hain?

👉 jab:

data transform karna ho

UI me show karne ke liye format change karna ho (Angular me common)

🔹 2. filter() kya hota hai?

👉 Condition ke basis pe elements select karta hai

Example:
let arr = [1, 2, 3, 4];

let even = arr.filter(function(num) {
  return num % 2 === 0;
});

console.log(even);

👉 Output:

[2, 4]
🔹 Use kab karte hain?

👉 jab:

data me se kuch elements chahiye ho

search, filtering UI (Angular forms)

🔹 3. reduce() kya hota hai? 🔥

👉 Pure array ko ek single value me convert karta hai

Example:
let arr = [1, 2, 3, 4];

let sum = arr.reduce(function(acc, curr) {
  return acc + curr;
}, 0);

console.log(sum);

👉 Output:

10

👉 acc = accumulator
👉 curr = current value

🔹 Use kab karte hain?

👉 jab:

sum nikalna ho

count karna ho

object banana ho

complex data transform

🔥 Difference Table (Very Important)
Feature	map()	filter()	reduce()
Return	New array	New array	Single value
Use	Transform	Select	Combine
Length	Same	<= original	1 value
🔹 Real Life Example 🧠
let users = [
  { name: "A", age: 20 },
  { name: "B", age: 25 }
];

// map
let names = users.map(u => u.name);

// filter
let adults = users.filter(u => u.age > 20);

// reduce
let totalAge = users.reduce((acc, u) => acc + u.age, 0);

console.log(names);     // ["A", "B"]
console.log(adults);    // [{name: "B", age:25}]
console.log(totalAge);  // 45
🔹 Interview Coding Questions 🔥
Q1️⃣ map()
let arr = [2, 3, 4];

let res = arr.map(x => x + 1);

console.log(res);

👉 Answer:

[3, 4, 5]
Q2️⃣ filter()
let arr = [10, 15, 20];

let res = arr.filter(x => x > 10);

console.log(res);

👉 Answer:

[15, 20]
Q3️⃣ reduce()
let arr = [1, 2, 3];

let res = arr.reduce((acc, curr) => acc * curr, 1);

console.log(res);

👉 Answer:

6
Q4️⃣ Tricky 🔥
let arr = [1, 2, 3];

let res = arr.map(x => {
  x * 2;
});

console.log(res);

👉 Answer:

[undefined, undefined, undefined]

👉 kyunki return missing hai

Q5️⃣ Combine use karo

👉 Question:
Even numbers ka square nikaalo

let arr = [1, 2, 3, 4];

let res = arr
  .filter(x => x % 2 === 0)
  .map(x => x * x);

console.log(res);

👉 Answer:

[4, 16]
Q6️⃣ reduce tricky 🔥
let arr = [1, 2, 3, 4];

let res = arr.reduce((acc, curr) => {
  if (curr % 2 === 0) {
    acc += curr;
  }
  return acc;
}, 0);

console.log(res);

👉 Answer:

6
🔹 Interview One-Line Answer

👉 Agar interviewer puche:

"Difference between map, filter, reduce?"

👉 bolo:

"map transforms each element, filter selects elements based on a condition, and reduce combines all elements into a single value."
