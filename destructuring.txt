Ye 4 concepts JavaScript me bahut important hain (almost har interview me puchte hain)

🔹 1. Array Destructuring kya hai?

👉 Array se values ko direct variables me nikalna

❌ Normal way:
const arr = [10, 20, 30];

const a = arr[0];
const b = arr[1];
✅ Destructuring way:
const arr = [10, 20, 30];

const [a, b, c] = arr;

console.log(a); // 10
console.log(b); // 20
🎯 Kab use hota hai?

Jab array se quickly values chahiye

API response handle karte time

Clean code ke liye

💡 Advanced Example:
const arr = [10, 20, 30, 40];

const [a, , c] = arr; // skip 20

console.log(a, c); // 10 30
🔹 2. Object Destructuring kya hai?

👉 Object ke properties ko direct variables me nikalna

❌ Normal way:
const user = {
  name: "Rahul",
  age: 25
};

const name = user.name;
✅ Destructuring:
const user = {
  name: "Rahul",
  age: 25
};

const { name, age } = user;

console.log(name); // Rahul
🎯 Kab use hota hai?

API response handle karte time

Angular/React me data bind karte time

Clean readable code ke liye

💡 Rename bhi kar sakte ho:
const { name: userName } = user;

console.log(userName);
🔹 3. Rest Operator (...rest) kya hai?

👉 Bachi hui values ko collect karta hai

📌 Array me:
const arr = [10, 20, 30, 40];

const [a, ...rest] = arr;

console.log(a);    // 10
console.log(rest); // [20, 30, 40]
📌 Object me:
const user = {
  name: "Rahul",
  age: 25,
  city: "Delhi"
};

const { name, ...rest } = user;

console.log(rest); 
// { age: 25, city: "Delhi" }
🎯 Kab use hota hai?

Remaining data collect karna ho

Dynamic data handle karna ho

Function arguments me

🔹 4. Spread Operator (...spread) kya hai?

👉 Array/Object ko expand (failana) karta hai

📌 Array copy / merge:
const arr1 = [1, 2];
const arr2 = [3, 4];

const result = [...arr1, ...arr2];

console.log(result); // [1,2,3,4]
📌 Object copy:
const user = { name: "Rahul" };

const newUser = { ...user, age: 25 };

console.log(newUser);
// { name: "Rahul", age: 25 }
🎯 Kab use hota hai?

Copy karna (immutability 🔥)

Merge karna

State update (React/Angular)

🔥 Rest vs Spread (INTERVIEW FAVORITE)

👉 Same syntax ... but use alag hai

Feature	Rest	Spread
Use	Collect karta hai	Expand karta hai
Example	[a, ...rest]	[...arr]
💯 Real Life Example (Angular/API)
const response = {
  id: 1,
  name: "Rahul",
  age: 25
};

const { name, ...rest } = response;

console.log(name); // Rahul
console.log(rest); // { id:1, age:25 }
🔥 Interview Coding Questions
✅ Q1: Swap 2 numbers using destructuring
let a = 10;
let b = 20;

[a, b] = [b, a];

console.log(a, b); // 20 10
✅ Q2: Find max using destructuring + spread
const arr = [10, 50, 30];

const max = Math.max(...arr);

console.log(max); // 50
✅ Q3: Remove property from object
const user = {
  name: "Rahul",
  age: 25,
  city: "Delhi"
};

const { age, ...rest } = user;

console.log(rest);
// { name: "Rahul", city: "Delhi" }
✅ Q4: Function me rest operator
function sum(...numbers) {
  return numbers.reduce((acc, curr) => acc + curr, 0);
}

console.log(sum(1,2,3,4)); // 10
🎯 Final Interview Line (yaad rakhna 🔥)

👉 “Destructuring helps extract values easily from arrays and objects.”
👉 “Rest operator collects remaining values.”
👉 “Spread operator expands elements for copying or merging.”
