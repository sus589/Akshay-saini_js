## in hinglish in javascript how function works example deke samjhao aur interview k liye coding question v dedo

Chalo JavaScript me function kaise kaam karta hai ekdum simple Hinglish + interview level tak samajhte hain 👇

🔹 Function kya hota hai?

👉 Function ek reusable block of code hota hai
👉 Ek baar likho, baar-baar use karo

🔹 Basic Syntax
function functionName() {
  // code
}
🔹 Simple Example
function greet() {
  console.log("Hello Susmita");
}

greet(); // function call

👉 Output:

Hello Susmita
🔹 Function ka flow kaise chalta hai?

Function define hota hai

Function call hota hai

JS execution context create karta hai

Code execute hota hai

Function finish → control wapas

🔹 Parameters & Arguments
function add(a, b) {
  console.log(a + b);
}

add(5, 3);

👉 Output:

8

👉 a, b = parameters
👉 5, 3 = arguments

🔹 Return Keyword
function multiply(a, b) {
  return a * b;
}

let result = multiply(4, 5);
console.log(result);

👉 Output:

20

👉 return value ko function ke bahar bhejta hai

🔹 Function Types 🔥
1. Function Declaration
function sayHi() {
  console.log("Hi");
}
2. Function Expression
const sayHello = function() {
  console.log("Hello");
};
3. Arrow Function (Modern JS)
const sum = (a, b) => a + b;

console.log(sum(2, 3));

👉 Output:

5
🔹 Execution Context (Interview Important)

👉 Jab function call hota hai:

JS ek new execution context banata hai

usme:

variables store hote hain

code execute hota hai

Example:

function test() {
  let x = 10;
  console.log(x);
}

test();

👉 yeh function apna alag memory space use karta hai

🔹 Real Life Example
function login(username) {
  return "Welcome " + username;
}

console.log(login("Susmita"));

👉 Output:

Welcome Susmita
🔹 Interview Coding Questions 🔥
Q1️⃣ Output kya hoga?
function test() {
  console.log("Hello");
}

test();

👉 Answer:

Hello
Q2️⃣ Output kya hoga?
function add(a, b) {
  return a + b;
}

console.log(add(2, 3));

👉 Answer:

5
Q3️⃣ Output kya hoga?
function demo(a, b) {
  console.log(a, b);
}

demo(5);

👉 Answer:

5 undefined

👉 kyunki second argument pass nahi hua

Q4️⃣ Tricky 🔥
function test() {
  return;
  console.log("Hello");
}

console.log(test());

👉 Answer:

undefined

👉 return ke baad code run nahi hota

Q5️⃣ Hoisting based
sayHi();

function sayHi() {
  console.log("Hi");
}

👉 Answer:

Hi
Q6️⃣ Arrow Function Tricky
const sum = (a, b) => {
  a + b;
};

console.log(sum(2, 3));

👉 Answer:

undefined

👉 kyunki return nahi likha

🔹 Interview One-Line Answer

👉 Agar interviewer puche:

"How functions work in JavaScript?"

👉 bolo:

"Functions in JavaScript are reusable blocks of code that execute when called. When a function is invoked, JavaScript creates a new execution context, runs the code inside it, and returns the result if specified."
